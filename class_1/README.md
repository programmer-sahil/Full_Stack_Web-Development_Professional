## 1. Class Objective

The objective of Class 1 is to build a strong foundation in HTML, CSS, and JavaScript before starting React.

Students will learn how a webpage is structured, styled, and made interactive. These three skills are required before learning React components, props, state, events, and list rendering.

By the end of this class, students should be able to create simple webpages, style them professionally, and add basic JavaScript logic.

## 2. What Students Will Learn

- How HTML creates the structure of a webpage
- How CSS controls layout, colors, spacing, and responsive design
- How JavaScript adds logic and interaction
- How to select elements from the DOM
- How events work in JavaScript
- How forms are handled using JavaScript
- How arrays and objects help organize data
- How `map()` and `filter()` prepare students for React list rendering
- How small real-world projects are planned and built

## 4. How to Run the Files

1. Open the `class_1` folder.
2. Choose any `.html` file.
3. Double-click the file to open it in a browser.
4. For the quiz project, open `mini_projects/05_class_1_quiz/index.html`.
5. No installation is required.

All examples use only HTML, CSS, and vanilla JavaScript.

## 5. Teaching Flow for the First Class

1. Start with basic HTML structure.
2. Explain common HTML tags using small examples.
3. Move to CSS and show how styling changes the same HTML.
4. Teach layout using box model, flexbox, grid, and responsive design.
5. Introduce JavaScript with variables and data types.
6. Explain conditions, loops, functions, arrays, and objects.
7. Connect JavaScript with the browser using DOM selection and events.
8. Practice form handling and localStorage.
9. Complete the mini projects.
10. End with the quiz and homework.

## 10. React Readiness Checklist

Before starting React, students should be comfortable with:

- Creating HTML structure
- Using classes and IDs
- Styling elements with CSS
- Building layouts with flexbox and grid
- Writing JavaScript variables and functions
- Working with arrays and objects
- Selecting DOM elements
- Handling click and submit events
- Rendering data from an array
- Understanding why reusable UI blocks are useful

## 11. Common Beginner Mistakes

- Forgetting to close HTML tags
- Using `class` and `id` without understanding the difference
- Writing CSS selectors incorrectly
- Forgetting semicolons in JavaScript
- Using `=` instead of `===` in conditions
- Forgetting to connect JavaScript to HTML
- Forgetting `event.preventDefault()` in forms
- Trying to learn React before understanding JavaScript basics

## 12. Practice Tasks for Students

- Create a responsive two-column layout.

## 15. Final Outcome After Class 1

After completing Class 1, students will understand how webpages are built from the ground up. They will know how HTML, CSS, and JavaScript work together and will be ready to start React with less confusion.

This class creates the foundation for React topics like components, props, state, event handling, conditional rendering, and list rendering.
